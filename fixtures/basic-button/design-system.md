# Basic Button Design System Fixture

## Tokens

- `--color-primary: #3b82f6`
- `--spacing-sm: 4px`
- `--spacing-md: 8px`
- `--spacing-lg: 16px`
- `--font-size-body: 16px`
- `--radius-sm: 4px`

## Rules

- Shared primitives must use semantic color tokens rather than hardcoded hex values.
- Spacing should use documented spacing tokens.
- Border radius should map to an approved radius token.
- Typography should follow the documented text scale.
